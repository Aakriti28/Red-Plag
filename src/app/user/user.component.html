Instructions

  1. Select a file to upload by clicking on the Browse button, and then click the Upload button.
  2. The User Files section contains all of the user's previous and newly uploaded files. Select any of these files to Download or Delete them by clicking on the buttons given below.
  3. The Boilerplate Code Files section contains the files which can be used as Boilerplate code as a reference when comparing multiple files.
  4. Select any file of type .zip/.tar.gz/.tar from the User Files section and click on Compare and Download Results to generate and download a pairwise comparison result of the files within.
  5. To remove any Boilerplate code from the files to be compared, Upload a separate .cpp/.py/.java file containing it and select it when it appears in the Boilerplate Code Files section, then click the Compare and Download Results button. This will remove the Boilerplate code when comparing the files.
